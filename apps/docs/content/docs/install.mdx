title: Install

---

ZK Freighter runs on the web, as a mobile app, and as a browser extension. You don't have to
install anything to try it — the [web wallet](https://app.zkfreighter.app) works in any browser.

> **Publishing status:** the app-store listings (Chrome Web Store, Microsoft Edge Add-ons, and a
> native iOS build) are **submitted and pending review**. You do not have to wait for them — every
> platform is installable right now via the direct links below.

> This is early, unaudited software. Testnet is the recommended surface.

**Looking for the download files?** Every build — the Android app, the iPhone app, and the
browser extension — lives on the
**[Releases page](https://github.com/Blockchain-Oracle/zk-freighter/releases/latest)**. The links
below point straight to the newest of each.

## Desktop — web wallet

Open **[app.zkfreighter.app](https://app.zkfreighter.app)** in any desktop browser. That's the
full wallet, nothing to install.

## Phone — mobile web app

The phone experience is a separate, thumb-first design at **[m.zkfreighter.app](https://m.zkfreighter.app)**.
It works on **any** phone (iPhone and Android) with no install and no store:

1. Open **[m.zkfreighter.app](https://m.zkfreighter.app)** in your phone's browser (Safari on
   iPhone, Chrome on Android).
2. Use the browser's **Share → Add to Home Screen**. It then launches full-screen like a native app.

(The desktop `app.zkfreighter.app` is built for large screens — on a phone it will point you here.)

## Android — native app (APK)

Prefer a real installed app? Install the signed APK directly, no Play Store account needed:

1. On your phone, download **[`zk-freighter.apk`](https://github.com/Blockchain-Oracle/zk-freighter/releases/latest/download/zk-freighter.apk)**
   (or scan the QR on [zkfreighter.app](https://zkfreighter.app)).
2. Android will ask **once** to allow installs from your browser — allow it.
3. Play Protect may offer to scan the app — normal for any app installed outside the Play Store;
   the wallet requests no SMS, contacts, or accessibility permissions.

**Auto-updates:** install [Obtainium](https://github.com/ImranR98/Obtainium) and point it at this
repository — it watches for new releases and updates the app for you (every release is signed with
the same key). A Google Play listing may follow later.

## iPhone

- **Now — the mobile web app.** Open **[m.zkfreighter.app](https://m.zkfreighter.app)** in Safari
  and **Add to Home Screen**. The full wallet, no install, every feature works.
- **Native app (AltStore).** If you have [AltStore](https://altstore.io) on your iPhone, add our
  source — AltStore then installs ZK Freighter and keeps it refreshed for you:
  in AltStore, **Browse → Sources → +** and paste **`https://zkfreighter.app/altstore.json`**.
  (Free-Apple-ID installs re-sign every 7 days; AltStore does this automatically. A paid Apple
  Developer account / TestFlight removes the limit and is planned.)

## Browser extension — Chrome, Edge, Brave

The extension adds a popup and side panel.

- **Chrome & Brave (recommended):** install from the
  **[Chrome Web Store](https://chromewebstore.google.com/detail/zk-freighter/gokjpinnopbmpmadbieeojfaehbmnjmd)** —
  one click, auto-updates. Brave installs straight from the Chrome Web Store too.
- **Edge:** the [Edge Add-ons](https://microsoftedge.microsoft.com/addons) listing is still in
  review — meanwhile Edge can also install from the Chrome Web Store link above.
- **Manual (any Chromium browser):** download the
  **[extension zip](https://github.com/Blockchain-Oracle/zk-freighter/releases/latest/download/zk-freighter-extension-chrome.zip)**,
  unzip it, open `chrome://extensions`, enable **Developer mode**, and choose **Load unpacked**.
